Active Directory • DHCP • GPO • FSRM • Network Design

Overview

This project involved building a full enterprise Windows Server environment including ADDS, OU structure, DHCP, GPOs, FSRM, and a domain-joined Windows 10 client.

Responsibilities

  • Configured DHCP scopes
  • Implemented 18 security-focused GPOs
  • Created shared folders & permissions
  • Configured client connectivity & testing

Skills Used

  • Active Directory
  • DHCP & DNS
  • Group Policy Management
  • FSRM & File Permissions
